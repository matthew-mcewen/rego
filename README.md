# Rego

## What is it?
- **Front End:** React Router
- **Back End:** C# .NET
- **Purpose:** A basic full-stack development demonstration.
- **Abstract:** The back-end service provides an API to query and filter a list of vehicles, and provides a SignalR webhook when a vehicle's registration is detected as expired. The front-end is a modern React web UI with responsive and mobile-first design principles.
